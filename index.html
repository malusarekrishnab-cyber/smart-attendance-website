Smart Attendance System

RFID & Firebase Based Attendance Management

About Project

Smart Attendance System is designed to automate the attendance process in colleges. This system reduces manual work, avoids proxy attendance, and provides accurate and secure attendance records for students and teachers.

Key Features

  • Attendance using RFID cards
  • Student can view own attendance
  • Teacher can view month-wise attendance
  • Automatic attendance calculation
  • No manual editing of data
  • Secure data storage using Firebase

RFID Working

  1. RFID card is scanned by the reader
  2. UID is read using Arduino / NodeMCU
  3. Data is sent to Firebase database
  4. Attendance is updated in Android application

Sample RFID Code (Reference Only)

#include <SPI.h>
#include <MFRC522.h>

#define SS_PIN 10
#define RST_PIN 9

MFRC522 rfid(SS_PIN, RST_PIN);

void setup() {
  Serial.begin(9600);
  SPI.begin();
  rfid.PCD_Init();
}

void loop() {
  if (!rfid.PICC_IsNewCardPresent()) return;
  if (!rfid.PICC_ReadCardSerial()) return;

  for (byte i = 0; i < rfid.uid.size; i++) {
    Serial.print(rfid.uid.uidByte[i], HEX);
  }
  Serial.println();
}
